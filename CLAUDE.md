# PhraseGen Project

A passphrase generator with Python CLI and web interface.

## Available Tools
- GitHub CLI (`gh`) is authenticated and available
- Use `gh` for all GitHub operations

## Project Structure
- `generate_passphrase.py` - Python CLI tool
- `index.html`, `passphrase.js` - Web interface
- `*.txt` - Word lists (adjectives, nouns, verbs) - all words must be exactly 4 letters

## Deployment
Hosted on GitHub Pages at https://devjourney.github.io/PhraseGen/
Push to main branch to auto-deploy.
